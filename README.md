## Overview
The **Job Portal Project** is a dynamic web application that connects **job seekers** with **employers**. The platform allows employers to post jobs, manage applications, and track applicant progress, while job seekers can browse jobs, apply, and manage their profiles.  

This project is built with **Laravel 10**, **Tailwind CSS**, and a MySQL database, offering a modern, responsive, and user-friendly interface.

---

## Features

### For Job Seekers
- Browse and search jobs with filters
- Apply to jobs directly
- Save jobs for later
- View application status
- Complete and update user profile
- Responsive dashboard with key stats

### For Employers
- Post and manage job listings
- View and filter applications
- Track job posting analytics
- Dashboard with job, application, and profile stats
- Manage company information

### Admin
- Manage roles and permissions
- Manage users, employers, and job seekers
- Oversee job postings and platform activity
- Dashboard for overall system monitoring
