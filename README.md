# JobConnect - Job Portal

## Overview
JobConnect is a modern job portal built with **Laravel 12** and **PHP 8.2**.  
It allows **job seekers** to browse and apply for jobs, and **employers** to post jobs and manage listings.  

---

## Features
- User roles: Admin, Employer, Job Seeker  
- Job & Company management  
- Request Employer Role  
- AI Chatbot using BotMan  
- Responsive UI with Tailwind CSS & Bootstrap  
- Newsletter subscription  

---

## Tech Stack
- **Backend:** Laravel 12, PHP 8.2  
- **Frontend:** Tailwind CSS, Bootstrap 5, FontAwesome  
- **Database:** MySQL / MariaDB (managed via **HeidiSQL**)  
- **Chatbot:** BotMan  
- **Rich Text Editor:** Trumbowyg
